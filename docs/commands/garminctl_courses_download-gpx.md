## garminctl courses download-gpx

Download course as GPX

### Synopsis

Download a course/route as a GPX file. Output goes to stdout by default, use -o to write to a file.

```
garminctl courses download-gpx <course_id> [flags]
```

### Options

```
  -h, --help            help for download-gpx
  -o, --output string   Output file path
```
